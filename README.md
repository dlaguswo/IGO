# IGO
국내 숙소 예약 홈페이지
- 핵심 기능
  * 회원가입 및 로그인(카카오톡, 네이버)
  * 숙소 검색 및 예약
  * 숙소 후기 리뷰 작성
  * 숙소 예약시 예약 문자 전송
  * 호스팅 신청을 통한 온라인 호스팅
  * Admin 페이지를 통한 숙소 관리
